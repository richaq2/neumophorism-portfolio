Website Link

Project detail

The website "Products Comparison" offers a straightforward and user-friendly platform for comparing various products. It allows users to easily evaluate different items based on features, prices, and other relevant criteria. With a clean interface, the site aims to simplify the decision-making process by presenting comparisons in a clear and organized manner, helping users make informed choices before making a purchase.

Technology and design complexity.

Tech used : HTML, CSS, ReactJS, Ant-Design, dummy json.

The web page includes a responsive Navbar with a logo and user profile picture, and a Sidebar with two menus: Product Details and Compare Products.The landing page contains a table built using Ant Design, which fetches product data from an API(dummy.json). It features pagination, sorting functionality, and an additional column with a "Compare Products" button for side-by-side comparison. Users can compare up to four products at a time based on attributes such as brand, features, price, and category. Duplicate products cannot be compared, and products already added to comparison disable the "Compare" button.Users can add more products to the comparison from a modal containing the product list in table format, with a restriction of four products for comparison at a time.
